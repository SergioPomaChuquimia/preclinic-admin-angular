# PreClinic — Panel de Administración Angular

## ¿De qué trata este proyecto?

Este es el panel de administración frontend del sistema de citas médicas PreClinic. Desarrollado con Angular, permite al personal de la clínica gestionar citas, ver el estado de las consultas y recibir notificaciones en tiempo real.

## Tecnologías utilizadas

- **Angular** — framework frontend
- **TypeScript** — lenguaje de programación

## Relación con otros repos

Este proyecto es el frontend del sistema. El backend y la API están en repos separados:
- `api-citas-medicas` — API REST en Laravel
- `sistema-citas-medicas-web` — sistema completo
